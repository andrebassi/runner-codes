Runner Codes is a **secure execution environment** for running code snippets generated by Large Language Models (LLMs) inside [Firecracker microVMs](https://firecracker-microvm.github.io/): secure, isolated, and fast.

## Features

**Security** - Full VM-level isolation using Firecracker microVMs, not just container namespaces. Each code execution runs in its own isolated environment with minimal attack surface.

**Performance** - Boot a microVM in ~125ms (cold) or ~70ms (warm with snapshots). ~5MB memory overhead per microVM.

**Multi-Language** - Support for 45+ programming languages including Python, Node.js, Go, Rust, Java, C/C++, and many more.

## Getting Started

```bash
# Download (Linux x86_64)
curl -L -o infra.operator https://github.com/andrebassi/runner-codes/releases/latest/download/infra.operator-linux-amd64
chmod +x infra.operator
sudo mv infra.operator /usr/local/bin/

# Setup (installs Firecracker, kernel, creates directories)
sudo infra.operator setup

# Execute code
sudo infra.operator host --lang python --code "print('Hello from Firecracker!')" --snapshot
```
