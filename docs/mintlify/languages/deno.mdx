# Deno

<Info>
**Versao**: Deno 1.39.0
**Rootfs**: 600 MB
**Tipo**: Runtime
</Info>

## Overview

Deno e um runtime seguro para JavaScript e TypeScript, criado pelo criador original do Node.js. Oferece TypeScript nativo, seguranca por padrao e ferramentas modernas integradas.

## Especificacoes

| Propriedade | Valor |
|-------------|-------|
| Versao | 1.39.0 |
| Comando | `deno run` |
| Extensao | `.ts` / `.js` |
| Rootfs Size | 600 MB |
| Tempo de Execucao | ~200ms |

## Exemplos

### Hello World
```typescript
console.log("Hello World");
```

### Variaveis Tipadas
```typescript

```

### Funcoes
```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

const add = (a: number, b: number): number => a + b;

console.log(greet("Deno"));
console.log(add(2, 3));
```

### Interfaces

### Async/Await
```typescript

  return new Promise(resolve => {
    setTimeout(() => resolve("Data loaded!"), 100);
  });
}

const data = await fetchData();
console.log(data);
```

## Diferencas do Node.js

- TypeScript nativo (sem configuracao)
- Imports via URL (nao usa npm)
- Seguranca por padrao (permissoes explicitas)
- APIs modernas (fetch, Web Crypto, etc)

## Limitacoes

- Sem acesso a rede (--allow-net desabilitado)
- Sem acesso a sistema de arquivos (--allow-read/write desabilitados)
- Timeout padrao de 10 segundos
- Memoria limitada a 512MB
